# 🌑 THE GILDED ABYSS (L'Abîme Doré)
**Bienvenue dans l'aventure ultime de l'école !** 

Ceci est un RPG 3D Low-Poly où chaque décision compte. Serez-vous un héros légendaire ou un vagabond de l'abîme ?

## 📜 Les Règles du Jeu
*   **Aventure :** Explorez le monde ouvert et découvrez des biomes (Forêts, Déserts, Cerisiers).
*   **La Mort :** Si vous mourrez, vous perdez **100 Pièces d'Or**.
*   **Sauvetage :** Pour 0.50€ (voir Sacha), vous pouvez récupérer votre or perdu.
*   **Niveau 3 :** Atteignez le niveau 3 pour pouvoir rejoindre une Guilde.

## ⚔️ Événements Spéciaux
*   **Samedi 20h :** Guerre des Guildes (Le Grand Choc).
*   **Dimanche (Midi) :** Tournoi de l'Abîme.

## 🚫 Charte de Bonne Conduite
Le système de modération est automatique. Tout propos raciste, homophobe ou sexiste entraînera un **Bannissement Progressif** :
1.  10h / 2. 20h / 3. 40h / 4. 80h / 5. 160h.
6.  **6ème ban :** RESET TOTAL du compte (Niveau 0) et ban d'un mois.

## 🛠️ Commandes
*   **Z, Q, S, D :** Se déplacer.
*   **K :** Menu Admin (Réservé à Sacha).
